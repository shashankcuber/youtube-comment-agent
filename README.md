# YouTube Comment Pulse

YouTube Comment Pulse is an AI-powered Chrome extension that analyzes YouTube comments and helps users decide whether a video is worth watching.

It adds a floating panel on YouTube video pages. When the user clicks **Analyze Comments**, the extension sends the video ID to a FastAPI backend. The backend fetches comments, analyzes public sentiment using a local LLM through Ollama, and returns a watch recommendation.

If the LLM fails, the system automatically falls back to a rule-based analysis pipeline.

---

## Features

- Chrome extension for YouTube video pages
- FastAPI backend
- YouTube comment fetching through YouTube Data API
- Local LLM support through Ollama, for example `qwen3:1.7b`
- Public sentiment analysis
- Watch / skim / skip recommendation
- Authenticity signal estimate from comments
- Positive, negative, neutral, and warning themes
- Rule-based fallback if the LLM fails
- Agent execution steps shown in the UI

## Environment Setup

Create `backend/.env`:

```env
YOUTUBE_API_KEY=your_youtube_api_key_here

LLM_PROVIDER=ollama
LLM_MODEL=qwen3:1.7b
OLLAMA_BASE_URL=http://localhost:11434
LLM_TIMEOUT_SECONDS=180
```

Do not commit `.env` to GitHub.

Recommended `.gitignore` entries:

```gitignore
backend/.env
.env
backend/venv/
__pycache__/
*.pyc
```

## Run Ollama

Install Ollama, then pull the model:

```bash
ollama pull qwen3:1.7b
```

Start Ollama:

```bash
ollama serve
```

Check that it is running:

```bash
curl http://localhost:11434/api/tags
```

---

## Run Backend

From the `backend/` folder:

```bash
uvicorn main:app --reload
```

Backend runs at:

```text
http://127.0.0.1:8000
```

Health check:

```text
http://127.0.0.1:8000/health
```

Analyze endpoint:

```text
http://127.0.0.1:8000/analyze?video_id=VIDEO_ID&max_comments=50&max_comments_for_llm=20
```

For `qwen3:1.7b`, recommended values are:

```text
max_comments=50
max_comments_for_llm=20
```

## Run as Chrome Extension

1. Open Chrome.
2. Go to:

```text
chrome://extensions/
```

3. Enable **Developer Mode**.
4. Click **Load unpacked**.
5. Select the folder:

```text
youtube-comment-agent/extension/
```

6. Open a YouTube video page:

```text
https://www.youtube.com/watch?v=...
```

7. Click **Analyze Comments** in the floating panel.

The extension will call the local backend and display the result.

## Output Shown in UI

The extension displays:

- Public sentiment
- Watch rating
- Watch decision
- Recommendation
- Authenticity signals
- Sentiment distribution
- LLM status
- Fallback status
- Agent execution steps

Example:

```text
Public Sentiment: Mostly Positive
Watch Rating: 8.2 / 10
Decision: Watch
Generated by: Local LLM qwen3:1.7b
```

If the LLM fails:

```text
Generated by: Rule-based fallback
LLM status: Failed
Error: Read timed out
```

## Common Issues

### Ollama connection refused

Start Ollama:

```bash
ollama serve
```

### Model not found

Pull the model:

```bash
ollama pull qwen3:1.7b
```

### LLM timeout

Increase timeout in `.env`:

```env
LLM_TIMEOUT_SECONDS=180
```

Or reduce:

```text
max_comments_for_llm=20
```

### Backend request failed

Make sure FastAPI is running:

```bash
uvicorn main:app --reload
```

### Extension panel not showing

Check that:

- You loaded the `extension/` folder
- You are on a YouTube video page
- You refreshed the YouTube page after loading the extension

---

## Notes

The system analyzes YouTube comments only. It does not watch the video, verify factual accuracy, or check external sources.

The authenticity score means **perceived authenticity based on comments**, not objective truth.
